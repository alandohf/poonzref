19.5 哪里可以找到兼容 ANSI 的 lint？

PC-Lint 和 FlexeLint 是 Gimpel Software 公司的 产品 (http://www.gimpel.com/)。

Unix System V 版本 4 的 lint 兼容 ANSI。 可以从 UNIX Support Labs 或 System V 的销售商 单独得到 (和其它 C 工具捆绑在一起)。

另外一个兼容 ANSI 的 lint 是 Splint (以前叫 lclint, http://www.splint.org/)。它可以 作一些高级别的正式检验。

如果没有 lint, 许多现代的编译器可以作出几乎和 lint 一样多的诊断。许多网友推荐 gcc -Wall -pedantic。
