19.15 谁有 C 编译器的测试套件？

Plum Hall (以前在 Cardiff, NJ, 现在在 Hawaii) 有一个套件出售; Ronald Guilmette 的 RoadTestTM 编译器测试套件 (更多 信息在  ftp://netcom.com/pub/rfg/roadtest/announce.txt);  Nullstone 的自动编译器性能分析工具 (http://www.nullstone.com)。  FSF 的 GNU C (gcc) 发布中含有一个许多编译器通常问题的 C 严酷测试。  Kahan 的偏执狂的测试 (ftp://netlib.att.com/netlib/paranoia), 尽其所能的测试 C 实现的浮点能力。
